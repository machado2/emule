Modificação em uma versão bem antiga do eMule, feita porque a forma como o eMule organizava a fila para dar mais prioridade para as pessoas que já nos forneceram alguma parte de algum arquivo. Desta forma ganhava "pontos" com os usuários que contribuiam de forma mais rápida, aumentando a velocidade dos downloads, além de beneficiar os outros usuários de forma mais justa.
